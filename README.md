## **Requirements**
* Swift 5.0 +
* XcodeGen 2.25.0 +
* Xcode 10.0 +

## **Getting started**
### Generate Xcode project

To generate Xcode project (`.xcodeproj`) file you must install [XcodeGen](https://github.com/yonaskolb/XcodeGen) and then run the following command in terminal

```bash
$ xcodegen
```
